Bootstrap

Bootstrap is a powerful, open-source framework for building responsive, mobile-first websites quickly. It provides ready-made components, utilities, and a grid system to make web development faster and easier.

Buttons

Cards

Flexible content containers with shadows, borders, and hover effects.

Navbar

Responsive navigation bars with collapse toggler.

Modals

Popups for alerts and extra content using .modal.

Accordion

Expandable content sections with .accordion.
